# 💰 Cash Crush

> Gamified Personal Finance & Money Management Platform

Cash Crush is a modern web application designed to help users build better financial habits through gamification, goal-based saving, budgeting, and financial tracking.

Instead of treating money management as a boring task, Cash Crush transforms it into an engaging journey where users can create goals, save money, earn achievements, track spending, and improve their financial discipline.

# ✨ Why Cash Crush is Different

Most finance applications focus only on expense tracking.

Cash Crush introduces a dual-saving philosophy:

### 🏦 Security Vault

Money reserved for emergencies and financial security.

### 🎯 Goal Savings

Money allocated toward dreams and aspirations.

This separation encourages users to build both financial stability and financial growth simultaneously.

Additionally, Cash Crush uses gamification mechanics such as:

- XP Rewards
- Level Progression
- Achievement Badges
- Saving Streaks
- Financial Challenges

to make money management engaging and habit-forming.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/your-username/cash-crush.git

cd cash-crush
```

## Install Frontend Dependencies

```bash
npm install
```

## Install Backend Dependencies

```bash
cd backend

npm install
```

---

# 🔐 Environment Variables

Create a `.env` file inside the backend directory.

```env
PORT=5000

DATABASE_URL=your_database_url

JWT_SECRET=your_jwt_secret

EMAIL_USER=your_email

EMAIL_PASSWORD=your_email_password

CLIENT_URL=http://localhost:5173
```

---

# ▶️ Running the Application

## Start Backend

```bash
cd backend

npm run dev
```

## Start Frontend

```bash
npm run dev
```

Application URLs:

```text
Frontend: http://localhost:5173

Backend: http://localhost:5000
```
